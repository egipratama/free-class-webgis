# Free Class WebGIS

Repositori ini berisi materi dasar untuk belajar WebGIS menggunakan HTML, CSS, dan JavaScript, serta contoh menampilkan peta menggunakan MAPID Maps.

## ğŸ“ Struktur Folder

- **/fundamental/**  
  Berisi contoh dasar penggunaan HTML, CSS, dan JavaScript.  
  Cocok untuk pemula yang ingin memahami struktur dan sintaks dasar web.

  **Materi di dalam folder ini meliputi:**
  - Dasar-dasar tag HTML (judul, paragraf, gambar, list, dsb)
  - Dasar penggunaan CSS (selector tag, class, id)
  - Dasar penggunaan JavaScript (event, manipulasi DOM)

- **/maps/**  
  Berisi contoh kode untuk menampilkan peta menggunakan MAPID Maps dan Maplibre GL JS.  
  Materi di dalamnya meliputi:
  - Cara menampilkan peta dasar
  - Menambahkan marker (titik), polyline (garis), dan polygon (area)
  - Tips Menampilkan data spasial dengan memanipulasi data basemap

## ğŸ“– Cara Menggunakan

1. **Buka folder `fundamental`**  
   Pelajari file-file HTML, CSS, dan JS untuk memahami dasar pembuatan web.

2. **Buka folder `maps`**  
   Lihat contoh kode untuk menampilkan peta dan menambahkan data spasial.

3. **Jalankan file HTML di browser**  
   Cukup klik dua kali file HTML atau buka melalui editor seperti VS Code dengan ekstensi Live Server.
